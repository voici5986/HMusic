# 赞赏功能设置指南

## 📱 功能说明

已为小爱音乐盒添加了温馨的赞赏功能，用户可以通过设置菜单访问赞赏页面。

## 🖼️ 添加您的赞赏码

### 步骤1：准备图片
1. 将您的微信赞赏码保存为 `sponsor_qr_code.png`
2. 建议尺寸：200x200 像素（或等比例）
3. 格式：PNG（推荐）或 JPG

### 步骤2：放置图片
将图片文件放置到以下位置：
```
assets/images/sponsor_qr_code.png
```

### 步骤3：删除占位符
删除临时占位符文件：
```
assets/images/sponsor_qr_placeholder.txt
```

## 🔧 技术实现

### 路由配置
- 路径：`/settings/sponsor`
- 菜单位置：设置 → 赞赏支持

### 图片处理

## 🎯 访问方式

用户可通过以下方式访问赞赏页面：
1. 主页右上角设置菜单
2. 点击 "赞赏支持" 选项
3. 进入精美的赞赏页面
